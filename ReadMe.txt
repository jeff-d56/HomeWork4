insertion sort with pq
takes any sequence and will sort it by inserting elements into a prioirty queue 
in sorted fashion and then just removing the first index back into the sequence

tree traversal
will travers any binary tree in pre_order and in_order
must be in this format

[root, left, right, left_L, left_R, right_L, right_R,...]

builds a tree like this based on entered list then traverses it in set orders
		root
	left		right
left_L	   left_R  right_L   right_R
